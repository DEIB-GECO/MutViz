TUMORS COMPARISON
The visualization allows to compare the distribution of mutations from different tumor-types around regions in two provided region-set through a heat-map. The x-axis represents the distance from the middle point of the provided regions, computed as (start+stop)/2, the y-axis discriminates the selected tumor-types and the color-intensity represents the amount of mutations falling in a given bin. The most opaque bin for a region set represents the bin containing the highest number of mutations. Use the toolbar on the left to select:
  • A region set, among the ones added to the workspace.
  • Multiple tumor types: each row of the heatmap shows the distribution of mutations for a given tumor type.
  • Mutation classes: only mutations belonging to the selected classes will be displayed.
  • bin size: number of base pairs per bin.
The horizontal slider helps restricting the domain.
The statistical test returns a p-value representing how different are the currently-displayed distributions. Only the selected mutation classes will be considered for testing.
